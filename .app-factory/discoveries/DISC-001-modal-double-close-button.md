# DISC-001 — Double croix de fermeture sur les modals

**Type:** BUG_UI
**Severity:** MINEUR
**Discovered during:** FIX-BLQ-001 validation
**Blocking:** Non

## Description
Les modals (ExpenseModal au minimum) affichent deux boutons de fermeture (×) superposés en haut à droite :
1. Un bouton "Fermer" sur le panneau/overlay
2. Un bouton × dans le header du modal à côté du titre
